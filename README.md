# -FARMERS-DISEASE-DAIGNOSTIC-AND-REPORTING-PORTAL-AI-BASED
Developed an AI-based Farmers Disease Diagnostic and Reporting Portal enabling farmers to upload crop images for instant disease detection. Implemented CNN models (ResNet, InceptionV3) using TensorFlow/PyTorch, with a React/Angular frontend, Django/Node.js backend, and scalable cloud deployment on AWS.
